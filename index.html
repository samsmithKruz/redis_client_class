RedisClient - PHP Redis Wrapper

RedisClient is a lightweight and developer-friendly PHP wrapper for Redis, designed to simplify interactions with Redis databases while providing full support for all Redis commands.

With RedisClient, developers can perform key-value storage operations, manage lists, sets, hashes, transactions, and even work with real-time messaging through Redis' publish-subscribe (Pub/Sub) system, all with an intuitive and easy-to-use interface.

Whether you're building a high-performance API, caching system, or real-time analytics, RedisClient ensures seamless integration with PHP applications, offering a robust and efficient solution for handling large-scale data operations.

Installation

To use RedisClient, ensure that Redis is installed on your system and that the PHP Redis extension is enabled.

1. Installing Redis

Windows

Download and install Redis for Windows from Microsoft Redis Releases. After installation, start the Redis server using: 

redis-server

Linux (Debian/Ubuntu)

Install Redis using: 

sudo apt update && sudo apt install redis-server

Start Redis with:

sudo systemctl start redis

MacOS

Install Redis via Homebrew:

brew install redis

Start Redis with:

brew services start redis

2. Installing the PHP Redis Extension

Ensure you have the PHP Redis extension installed. Use the following command:

For Linux/MacOS (Using PECL)

sudo pecl install redis

Then add the extension to your PHP configuration:

echo "extension=redis.so" >> $(php -i | grep -o '/.*php.ini')

For Windows

Download the appropriate Redis extension DLL, move it to your PHP `ext` directory, and enable it in `php.ini` by adding: 

extension=php_redis.dll

3. Installing RedisClient

To install RedisClient, simply include it in your project:

Using Composer

composer require your-namespace/redisclient

Manual Installation

Download the RedisClient class and include it in your project:

<?php
require 'path/to/RedisClient.php';
$redis = new RedisClient();
?>

RedisClient is now ready to use!

Usage

Learn how to use RedisClient in your projects with simple and advanced operations.

1. Initializing RedisClient

First, create an instance of RedisClient:

<?php
require 'RedisClient.php';

$redis = new RedisClient();
?>

2. Basic CRUD Operations

Insert a Key-Value Pair

$redis->insert('username', 'john_doe');

Retrieve a Value

$username = $redis->select('username');
echo $username; // Output: john_doe

Update a Value

$redis->update('username', 'jane_doe');

Delete a Key

$redis->delete('username');

3. Increment & Decrement

Increment a Counter

$redis->insert('counter', 1);
$redis->increment('counter');
echo $redis->select('counter'); // Output: 2

Decrement a Counter

$redis->decrement('counter');
echo $redis->select('counter'); // Output: 1

4. Working with Lists

Push & Pop Elements

$redis->push('users', 'Alice');
$redis->push('users', 'Bob');
echo $redis->pop('users'); // Output: Alice

5. Hash Operations

Insert and Retrieve Hash Fields

$redis->hash_insert('user:1', 'name', 'John');
echo $redis->hash_select('user:1', 'name'); // Output: John

Retrieve All Hash Fields

print_r($redis->hash_select_all('user:1'));

6. Working with Sets

Add and Retrieve Set Members

$redis->set_add('admins', 'alice');
$redis->set_add('admins', 'bob');
print_r($redis->set_members('admins'));

7. Working with Sorted Sets

Add and Retrieve Sorted Members

$redis->zadd('leaderboard', 100, 'Alice');
$redis->zadd('leaderboard', 150, 'Bob');
print_r($redis->zrange('leaderboard', 0, -1, true)); // Sorted by score

8. Executing Transactions

$redis->transaction(function($r) {
    $r->insert('key1', 'value1');
    $r->insert('key2', 'value2');
});

9. Setting Expiry & TTL

$redis->insert('temp', 'data');
$redis->set_expiration('temp', 30); // Expires in 30 seconds
echo $redis->get_ttl('temp');

10. Publish/Subscribe

Publishing a Message

$redis->publish('news', 'Breaking News: Redis is awesome!');

Subscribing to a Channel

$redis->subscribe('news', function ($channel, $message) {
    echo "Received on $channel: $message";
});

11. Flushing & Cleanup

$redis->flush(); // Clears all Redis data
$redis->close(); // Closes the connection

This covers all the major Redis operations available in RedisClient.

Commands

Detailed command list for RedisClient.

1. Key Operations

  • insert(key, value) - Set a key with a value.
  • select(key) - Get the value of a key.
  • update(key, value) - Update the value of an existing key.
  • delete(key) - Remove a key.
  • exists(key) - Check if a key exists.
  • rename(oldKey, newKey) - Rename a key.
  • set_expiration(key, seconds) - Set expiration for a key.
  • get_ttl(key) - Get the time-to-live (TTL) of a key.

2. String Operations

  • append(key, value) - Append a value to a key.
  • get_range(key, start, end) - Get a substring of a string key.
  • strlen(key) - Get the length of a string key.

3. Increment & Decrement

  • increment(key, amount = 1) - Increment a key’s value.
  • decrement(key, amount = 1) - Decrement a key’s value.

4. List Operations

  • push(key, value) - Push an element to a list.
  • pop(key) - Pop an element from a list.
  • lrange(key, start, stop) - Get elements from a list.
  • llen(key) - Get the length of a list.

5. Hash Operations

  • hash_insert(hash, field, value) - Insert a value into a hash field.
  • hash_select(hash, field) - Get a value from a hash.
  • hash_select_all(hash) - Get all fields and values from a hash.
  • hash_delete(hash, field) - Delete a field from a hash.
  • hash_exists(hash, field) - Check if a hash field exists.

6. Set Operations

  • set_add(key, value) - Add a member to a set.
  • set_members(key) - Get all members of a set.
  • set_remove(key, value) - Remove a member from a set.
  • set_exists(key, value) - Check if a value exists in a set.

7. Sorted Set Operations

  • zadd(key, score, value) - Add a member with a score to a sorted set.
  • zrange(key, start, stop, withScores = false) - Get members in a score range.
  • zrem(key, value) - Remove a member from a sorted set.
  • zcount(key, min, max) - Count members within a score range.

8. Transactions

  • transaction(callback) - Execute multiple commands in a transaction.

9. Publish/Subscribe (Pub/Sub)

  • publish(channel, message) - Publish a message to a channel.
  • subscribe(channel, callback) - Subscribe to a channel.

10. Server & Maintenance Commands

  • flush() - Clear all data from Redis.
  • ping() - Check if the Redis server is responsive.
  • close() - Close the Redis connection.

This list covers all supported Redis commands in RedisClient.

Examples

Code examples demonstrating RedisClient functionality.

1. Connecting to Redis

        
        require 'RedisClient.php';

        // Initialize RedisClient
        $redis = new RedisClient('127.0.0.1', 6379);

2. Setting and Getting Data

        
        // Store a value
        $redis->insert('username', 'john_doe');

        // Retrieve a value
        echo $redis->select('username'); // Output: john_doe

3. Working with Lists

        
        // Push elements into a list
        $redis->push('users', 'Alice');
        $redis->push('users', 'Bob');

        // Retrieve list elements
        print_r($redis->lrange('users', 0, -1)); // Output: ['Alice', 'Bob']

4. Hashes - Storing Objects

        
        // Insert user data into a hash
        $redis->hash_insert('user:1001', 'name', 'Alice');
        $redis->hash_insert('user:1001', 'age', '25');

        // Retrieve a field value
        echo $redis->hash_select('user:1001', 'name'); // Output: Alice

        // Retrieve all fields
        print_r($redis->hash_select_all('user:1001'));

5. Using Sets

        
        // Add values to a set
        $redis->set_add('fruits', 'apple');
        $redis->set_add('fruits', 'banana');

        // Get all set members
        print_r($redis->set_members('fruits')); // Output: ['apple', 'banana']

6. Sorted Sets (Leaderboard Example)

        
        // Add scores to a sorted set
        $redis->zadd('leaderboard', 100, 'Player1');
        $redis->zadd('leaderboard', 150, 'Player2');

        // Get top players
        print_r($redis->zrange('leaderboard', 0, -1, true));

7. Transactions

        
        $redis->transaction(function($tx) {
            $tx->insert('balance', 100);
            $tx->increment('balance', 50);
        });

        echo $redis->select('balance'); // Output: 150

8. Publish/Subscribe

        
        // Publish a message
        $redis->publish('news_channel', 'Breaking News!');

        // Subscribe to a channel (example inside a loop)
        $redis->subscribe('news_channel', function($channel, $message) {
            echo "Received on $channel: $message";
        });

9. Clearing Redis

        
        // Delete a single key
        $redis->delete('username');

        // Flush the entire database
        $redis->flush();
